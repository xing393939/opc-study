#### 常用函数
| 函数               | 积分                  |
|------------------|---------------------|
| $∫x^n dx$        | $x^{n+1}/(n+1) + C$ |
| $∫\sin x dx$     | $-\cos x + C$       |
| $∫\cos x dx$     | $\sin x + C$        |
| $∫\rm e^{cx} dx$ | $\rm e^{cx}/c + C$  |
| $∫\ln x dx$      | $x\ln x - x + C$    |
